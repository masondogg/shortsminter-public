Allow ShortsMinter Through
Windows Defender Firewall

Windows sometimes blocks new apps by default. This quick guide walks you through adding ShortsMinter to your firewall's allowed list — no advanced settings needed.

🪟 Windows 10 / 11
⏱ 2–3 minutes
🎯 ShortsMinter.exe
1
Find Your ShortsMinter.exe
Locate the application on your desktop or in the install folder — you'll need to know where it is for Step 7.

Find the ShortsMinter icon on your desktop. You'll need the exact file path for Step 7 — the quickest way to get it is to right-click the icon → Properties → Open File Location. The folder that opens is where ShortsMinter.exe lives.

📁

Default install path: Unless you changed it during installation, ShortsMinter is located at C:\Program Files\Masondogg Studios\ShortsMinter\ShortsMinter.exe — keep that handy for Step 7.

You can also launch ShortsMinter first — if Windows shows a firewall prompt when the app opens, dismiss it for now and come back to this guide. That pop-up is what we're here to fix.

2
Open Windows Security
Navigate to Windows Security from the Start menu or system tray.

Press the Windows key on your keyboard, type "Windows Security", and press Enter — or click the top result when it appears. You can also click the shield icon in your system tray (bottom-right of the taskbar).

3
Go to Firewall & Network Protection
Inside Windows Security, locate the Firewall section.

In the Windows Security home screen, click Firewall & network protection in the left sidebar. You'll land on a page showing your network profiles — domain, private, and public.

4
Click "Allow an app through firewall"
Scroll down to find the link that opens the allowed apps list.

Scroll down on the Firewall & network protection page until you see Allow an app through firewall. Click it. This is the direct route — no need to go into Advanced Settings.

5
Click "Change Settings"
The allowed apps list opens. You need to unlock it before you can make changes.

The Allowed apps window opens showing all currently permitted apps. The list will be greyed out and locked. Click Change settings in the upper-right — Windows may ask for admin confirmation, click Yes.

Once unlocked, the list becomes interactive and the Allow another app… button at the bottom activates.

6
Click "Allow another app…"
With settings unlocked, add ShortsMinter to the allowed list.

With the list now unlocked, scroll to the bottom of the window and click Allow another app… This opens a file browser dialog so you can point Windows to ShortsMinter.

7
Browse to ShortsMinter.exe
Use the file browser to locate the ShortsMinter application file.

An Add an app dialog appears. Click Browse… and navigate to the ShortsMinter executable. The default install path is:

C:\Program Files\Masondogg Studios\ShortsMinter\ShortsMinter.exe
🖱️

Can't find it? Right-click the ShortsMinter desktop icon → Properties → Open File Location. The folder that opens is where ShortsMinter.exe lives. Paste that folder path into the Browse dialog's address bar, then select ShortsMinter.exe.

Once ShortsMinter.exe is selected and the path is shown, click Add.

8
Check Private & Public — Then Click OK
ShortsMinter is now in the list. Make sure both network types are checked.

ShortsMinter now appears in the allowed apps list. You'll see two checkboxes to its right — one for Private and one for Public networks. Make sure both are checked.

Click OK to save and close the window. That's it — ShortsMinter is now allowed through your firewall. ✅

💡

Why both? Private covers your home or office Wi-Fi. Public covers hotspots and other networks. Checking both means ShortsMinter works wherever you are without needing to revisit this screen.

9
Confirm & You're Done
Verify ShortsMinter is in the allowed list, then close and launch the app.

Close the Allowed Apps window. ShortsMinter is now on the approved list and Windows Defender Firewall will no longer block it.

Launch ShortsMinter from your desktop — it should open and operate normally without any firewall interruption or security prompts.

🎯

Still seeing a prompt? If Windows pops up a firewall alert when you launch ShortsMinter, click Allow access and both checkboxes (Private + Public) will be confirmed automatically — same result, different path.
