GETTING STARTED

How to install ShortsMinter

  1. Download the installer from your Gumroad purchase confirmation email. The file is named ShortsMinter_Setup_v1.0b.exe

  2. Run the installer. Windows may show a security warning — click More info then Run anyway. This is normal for new independent software.

  3. Follow the install wizard. The runtime (~400MB) installs automatically on first install. This is a one-time download.

  4. Launch ShortsMinter from your Start Menu or desktop shortcut.

  5. Enter your license key on first launch. Your key was in your Gumroad purchase confirmation email.

System requirements: Windows 10/11 64-bit · NVIDIA GPU recommended · 8GB RAM minimum · 10GB free storage
